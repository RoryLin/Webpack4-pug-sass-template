# webpack4 with pug and sass

clone and run:
- development mode with hot reload
>npm i && npm run start:dev
- development mode
>npm i && npm run dev
- production mode
>npm i && npm run prod

  - [index.js](src/index.js)
  - [index.pug](src/index.pug)
  - __sass__
